Kotlin

Variables

In Kotlin, you can declare variables using val (immutable) and var (mutable).

val name = "Sharn" // Immutable variable
var age = 20 // Mutable variable

Explicitly Declaring Data Types

Kotlin allows you to explicitly declare data types.

val number: Int = 10
val pi: Double = 3.14

Null Variables

In Kotlin, variables can be nullable using ?.

var nullableString: String? = null

Lateinit and By Lazy

Using lateinit for variables that will be initialized later:

lateinit var name: String
name = "Sharn"

Using by lazy for variables that are initialized when first accessed:

val lazyValue: String by lazy {
    "This is a lazy initialization"
}

Methods and Functions

Defining and calling functions in Kotlin:

fun greet() {
    println("Hello, Kotlin!")
}
greet()

Companion Object

Using companion objects in Kotlin:

class MyClass {
    companion object {
        fun myFunction() {
            println("Hello from Companion Object")
        }
    }
}
MyClass.myFunction()
