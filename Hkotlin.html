Kotlin

Variables

Kotlin me variables do tarike se declare kiye jate hain: val (jo badal nahi sakta) aur var (jo badal sakta hai).

val name = "Sharn" // Fix variable
var age = 20 // Change hone wala variable

Explicitly Declaring Data Types

Kotlin me tum directly data type define kar sakte ho.

val number: Int = 10
val pi: Double = 3.14

Null Variables

Kotlin me kisi variable ko null hone dena ho toh ? lagana padta hai.

var nullableString: String? = null

Lateinit aur By Lazy

lateinit ka use jab hota hai jab variable ko baad me initialize karna ho.

lateinit var name: String
name = "Sharn"

by lazy ka use jab hota hai jab variable pehli baar access hone par initialize ho.

val lazyValue: String by lazy {
    "Ye ek lazy initialization hai"
}

Methods aur Functions

Kotlin me function define aur call kaise karte hain:

fun greet() {
    println("Hello, Kotlin!")
}
greet()

Companion Object

Companion object ka use:

class MyClass {
    companion object {
        fun myFunction() {
            println("Hello from Companion Object")
        }
    }
}
MyClass.myFunction()

Kotlin Classes

Kotlin me class kaise define karte hain:

class Person(val name: String, val age: Int) {
    fun greet() {
        println("Hello, my name is $name and I am $age years old.")
    }
}

val p = Person("Sharn", 20)
p.greet()
